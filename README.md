Librava is a lightweight library management platform built with PHP 8+ using a custom MVC architecture.
It provides a RESTful API designed for both Web and Android applications.

## Features
- Custom MVC (No framework)
- RESTful API
- Token-based authentication
- Books & Borrow management
- Pagination & search
- Multilingual (FA / EN)
- RTL / LTR support
- Bootstrap 5 UI
